# Brain Over Brawn ⚽

A training guide for center backs who play with brains over brawn. Curated videos, drills, and a shareable playlist system.

## Deploy to Vercel (5 minutes)

### Step 1: Create a GitHub account (skip if you have one)
Go to [github.com](https://github.com) and sign up for free.

### Step 2: Create a new repository
1. Click the **+** button (top right) → **New repository**
2. Name it `brain-over-brawn`
3. Keep it **Public**
4. Click **Create repository**

### Step 3: Upload these files
1. On your new repo page, click **"uploading an existing file"**
2. Drag the entire contents of this folder into the upload area
3. Click **Commit changes**

### Step 4: Deploy on Vercel
1. Go to [vercel.com](https://vercel.com) and sign up with your GitHub account
2. Click **Add New → Project**
3. Find `brain-over-brawn` in the list and click **Import**
4. Vercel auto-detects Vite — just click **Deploy**
5. Wait ~60 seconds — done! You'll get a URL like `brain-over-brawn.vercel.app`

### Step 5: Share with your kid
Send them the URL. They can add it to their phone home screen:
- **iPhone**: Open in Safari → Share → Add to Home Screen
- **Android**: Open in Chrome → Menu → Add to Home Screen

## How the Share Feature Works
1. Add videos to your playlist with tags and notes
2. Tap **Share My Playlist** → copy the code
3. Text/email the code to your kid
4. They tap **Import Videos** → paste the code → done
